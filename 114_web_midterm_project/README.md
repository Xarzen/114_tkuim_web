# 2025 英雄聯盟世界賽決賽資訊網

## 專案資訊

- 專案名稱：2025 LOL Worlds Finals Info
- 製作者：411630683 雷天佑 （一人一組）

## 專案簡介

本專案為 2025 英雄聯盟世界賽總決賽的完整資訊平台，提供比賽結果展示、投票預測、詳細賽事資訊等功能。網站使用響應式設計，完美支援桌面、平板、手機等各種裝置。

## 使用技術

### 前端技術
- **HTML5**：語意化標籤、表單元素
- **CSS3**：Flexbox 佈局、Grid 系統、動畫效果、媒體查詢（RWD）
- **Bootstrap 5.3.0**：響應式網格系統、導覽列、工具類別
- **Bootstrap Icons 1.11.1**：圖示庫
- **JavaScript (ES6+)**：DOM 操作、事件處理、localStorage 應用

### 核心技術應用
- **響應式網頁設計（RWD）**：完整支援各種螢幕尺寸
- **DOM 操作**：`querySelector`、`addEventListener`
- **localStorage**：投票資料持久化儲存、表單送出記錄
- **HTML5 Constraint Validation API**：表單驗證（`checkValidity()`、`validity` 屬性）
- **Web Notification API**：瀏覽器原生通知系統
- **CSS 動畫**：過渡效果、漸層色、hover 互動
- **錨點導航**：平滑滾動至各區塊
- **mailto 協定**：自動開啟郵件程式

## 網站功能特色

### 1. Hero Section（首頁橫幅）
- 背景圖片與深色遮罩設計
- 顯示比賽時間與地點資訊
- 大標題與粗體文字設計
- 完整的視覺層次

### 2. 比賽結果展示
- 大比分顯示：T1 3:2 KT
- 左右對抗式排版：兩隊分居左右，中間顯示比分
- 隊伍 Logo 點擊可連結至 Instagram 官方帳號
- 地區標籤顯示（LCK）

### 3. 投票預測系統
- 左右對抗式投票介面：T1 vs KT
- 隊伍 Logo 展示與投票按鈕
- localStorage 防重複投票：每位訪客僅能投票一次
- 即時結果顯示：
  - 投票數量統計
  - 百分比計算
  - 動態進度條（帶漸層色彩）
  - 總投票數顯示
- 投票後顯示感謝訊息

### 4. 賽事詳情（各局比賽）
- 五局比賽完整資訊（BO5）
- 下拉式卡片設計：點擊展開/收合
- 左右對抗式比分顯示：
  - 左側：T1 隊名 + 勝敗狀態
  - 中間：局數、時間、VS
  - 右側：勝敗狀態 + KT 隊名
- 每局包含：
  - 比賽資訊：比賽時長、擊殺數
  - 關鍵數據：總經濟、摧毀防禦塔、擊殺小龍、擊殺遠古巨龍、擊殺巴龍、擊殺預示者、擊殺虛空幼蟲
  - YouTube 影片嵌入（實際比賽影片）
- 決勝局特殊標記：使用文字符號標示

### 5. 聯絡表單
- HTML5 表單驗證：使用 Constraint Validation API
- 必填欄位驗證：
  - Email：必填 + Email 格式驗證
  - 主旨：必填 + 最少 3 個字元
  - 訊息內容：必填 + 最少 10 個字元
- 即時驗證回饋：
  - 綠色外框與勾選圖示（驗證通過）
  - 紅色外框與驚嘆號圖示（驗證失敗）
  - 自訂錯誤訊息顯示
- 表單送出處理：
  - localStorage 儲存送出記錄
  - 自動開啟郵件程式（mailto）
  - 瀏覽器通知提示送出成功
- 防止頁面重新載入：多層防護機制

### 6. 網頁通知（Web Notification API）
- 自動請求通知權限：首次表單送出時
- 通知內容：
  - 標題：「訊息已送出」
  - 內容：「我們已收到您的訊息，謝謝！」
  - 圖示：T1 隊伍 Logo
  - 震動效果：支援行動裝置
- 瀏覽器原生通知：顯示於系統通知中心
- 權限管理：遵循使用者隱私設定

### 7. 導覽列與頁尾
- 固定式導覽列（Sticky Navbar）
- 五個錨點連結：首頁、比賽結果、投票預測、賽事詳情、聯絡我們
- 平滑滾動效果
- 頁尾快速連結：提供相同的錨點導航
- 版權資訊與製作者資訊

### 8. 使用者體驗優化
- 完整的 Hover 互動效果
- 平滑的動畫與過渡
- 清晰的視覺層次
- 一致的配色系統
- 使用文字符號代替 emoji
